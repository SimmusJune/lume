## 2) Publish on GitHub Pages

If your repo name is `yourname.github.io`, place these files at repo root or copy this folder contents to root.

If this is a project repo (for example `lume-pages`), keep files in root and enable Pages:

1. Go to GitHub repo `Settings -> Pages`
2. Source: `Deploy from a branch`
3. Branch: `main` (or `master`), folder: `/ (root)`
4. Save and wait for deployment

Your URLs will be:

- `https://<username>.github.io/<repo>/support/`
- `https://<username>.github.io/<repo>/privacy/`

## 3) Fill App Store Connect

- `Support URL`: support page URL
- `Privacy Policy URL`: privacy page URL
